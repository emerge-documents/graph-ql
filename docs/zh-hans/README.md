# GraphQL

工作草案 – 2016 十月

## 介绍

「GraphQL」是「Facebook」在2012年创建的一种用于描述客户端到服务器应用程序的数据模型的功能和要求查询语言RFC规范草案。该标准的制定始于2015年。「GraphQL」是一种在演变中的新语言。 在本规范的未来版本中将继续进行显着的增强。
